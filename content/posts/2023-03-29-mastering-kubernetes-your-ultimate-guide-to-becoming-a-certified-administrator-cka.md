        <p id="1095" class="pw-post-body-paragraph pb pc oc pe b pf pg df ph pi pj dj pk py pm pn po pz pq pr ps qa pu pv pw px jl bi" data-selectable-paragraph="">
          <em class="pd">CKA or Certified Kubernetes Administrator is one of the key certifications in the DevOps journey.</em>
        </p>
        
        <p id="5702" class="pw-post-body-paragraph pb pc oc pe b pf pg df ph pi pj dj pk py pm pn po pz pq pr ps qa pu pv pw px jl bi" data-selectable-paragraph="">
          As per the landing page for the <a class="ae lw" href="https://training.linuxfoundation.org/certification/certified-kubernetes-administrator-cka/" target="_blank" rel="noopener ugc nofollow">CKA</a> on Linux Foundation.
        </p>
        
        <blockquote class="oy oz pa">
          <p id="5cf4" class="pb pc pd pe b pf pg df ph pi pj dj pk pl pm pn po pp pq pr ps pt pu pv pw px jl bi" data-selectable-paragraph="">
            CKA was created by The Linux Foundation and the Cloud Native Computing Foundation (CNCF) as a part of their ongoing effort to help develop the Kubernetes ecosystem. The exam is an online, proctored, performance-based test that requires solving multiple tasks from a command line running Kubernetes.
          </p>
        </blockquote>
        
        <p id="6f32" class="pw-post-body-paragraph pb pc oc pe b pf pg df ph pi pj dj pk py pm pn po pz pq pr ps qa pu pv pw px jl bi" data-selectable-paragraph="">
          The objective, as mentioned on the Linux Foundation website simply mentions:
        </p>
        
        <blockquote class="oy oz pa">
          <p id="31d4" class="pb pc pd pe b pf pg df ph pi pj dj pk pl pm pn po pp pq pr ps pt pu pv pw px jl bi" data-selectable-paragraph="">
            A certified K8s administrator has demonstrated the ability to do <strong class="pe qr">basic installation </strong>as well as <strong class="pe qr">configuring and managing production-grade Kubernetes clusters</strong>. They will have an understanding of key concepts such as Kubernetes networking, storage, security, maintenance, logging and monitoring, application lifecycle, troubleshooting, API object primitives and the ability to establish basic use-cases for end users.
          </p>
        </blockquote>
        
        <p id="a009" class="pw-post-body-paragraph pb pc oc pe b pf pg df ph pi pj dj pk py pm pn po pz pq pr ps qa pu pv pw px jl bi" data-selectable-paragraph="">
          <strong class="pe qr">Important points to Note</strong>
        </p>
        
        <ul class="">
          <li id="d6e9" class="vb vc oc pe b pf pg pi pj py vd pz ve qa vf px vg vh vi vj bi" data-selectable-paragraph="">
            The exam is based on <strong class="pe qr">Kubernetes v1.23</strong>.
          </li>
          <li id="6cdd" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            For the CKA Exam, a score of 66% or above must be earned to pass.
          </li>
          <li id="f569" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            CKA and CKAD Certifications are valid for 3 years.
          </li>
        </ul>
        
        <p id="58bb" class="pw-post-body-paragraph pb pc oc pe b pf pg df ph pi pj dj pk py pm pn po pz pq pr ps qa pu pv pw px jl bi" data-selectable-paragraph="">

          <strong class="pe qr">Objectives</strong>
        </p>
        
        <p id="1c52" class="pw-post-body-paragraph pb pc oc pe b pf pg df ph pi pj dj pk py pm pn po pz pq pr ps qa pu pv pw px jl bi" data-selectable-paragraph="">
          Storage 10%
        </p>
        
        <ul class="">
          <li id="112d" class="vb vc oc pe b pf pg pi pj py vd pz ve qa vf px vg vh vi vj bi" data-selectable-paragraph="">
            Understand storage classes, persistent volumes
          </li>
          <li id="6441" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Understand volume mode, access modes and reclaim policies for volumes
          </li>
          <li id="fbca" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Understand persistent volume claims primitive
          </li>
          <li id="2616" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Know how to configure applications with persistent storage
          </li>
        </ul>
        
        <p id="3286" class="pw-post-body-paragraph pb pc oc pe b pf pg df ph pi pj dj pk py pm pn po pz pq pr ps qa pu pv pw px jl bi" data-selectable-paragraph="">
          Troubleshooting 30%
        </p>
        
        <ul class="">
          <li id="f2f0" class="vb vc oc pe b pf pg pi pj py vd pz ve qa vf px vg vh vi vj bi" data-selectable-paragraph="">
            Evaluate cluster and node logging
          </li>
          <li id="2659" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Understand how to monitor applications
          </li>
          <li id="a7df" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Manage container stdout & stderr logs
          </li>
          <li id="2123" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Troubleshoot application failure
          </li>
          <li id="7773" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Troubleshoot cluster component failure
          </li>
          <li id="cf97" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Troubleshoot networking
          </li>
        </ul>
        
        <p id="6a81" class="pw-post-body-paragraph pb pc oc pe b pf pg df ph pi pj dj pk py pm pn po pz pq pr ps qa pu pv pw px jl bi" data-selectable-paragraph="">
          Workloads & Scheduling 15%
        </p>
        
        <ul class="">
          <li id="5dfd" class="vb vc oc pe b pf pg pi pj py vd pz ve qa vf px vg vh vi vj bi" data-selectable-paragraph="">
            Understand deployments and how to perform rolling update and rollbacks
          </li>
          <li id="9288" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Use ConfigMaps and Secrets to configure applications
          </li>
          <li id="c396" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Know how to scale applications<br /> Understand the primitives used to create robust, self-healing, application deployments
          </li>
          <li id="f2ef" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Understand how resource limits can affect Pod scheduling
          </li>
          <li id="a7c1" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Awareness of manifest management and common templating tools
          </li>
        </ul>
        
        <p id="d69f" class="pw-post-body-paragraph pb pc oc pe b pf pg df ph pi pj dj pk py pm pn po pz pq pr ps qa pu pv pw px jl bi" data-selectable-paragraph="">
          Cluster Architecture, Installation & Configuration 25%
        </p>
        
        <ul class="">
          <li id="6b19" class="vb vc oc pe b pf pg pi pj py vd pz ve qa vf px vg vh vi vj bi" data-selectable-paragraph="">
            Manage role based access control (RBAC)
          </li>
          <li id="1403" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Use Kubeadm to install a basic cluster
          </li>
          <li id="fae8" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Manage a highly-available Kubernetes cluster
          </li>
          <li id="647a" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Provision underlying infrastructure to deploy a Kubernetes cluster
          </li>
          <li id="a203" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Perform a version upgrade on a Kubernetes cluster using Kubeadm
          </li>
          <li id="a3c5" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Implement etcd backup and restore
          </li>
        </ul>
        
        <p id="f7a7" class="pw-post-body-paragraph pb pc oc pe b pf pg df ph pi pj dj pk py pm pn po pz pq pr ps qa pu pv pw px jl bi" data-selectable-paragraph="">
          Services & Networking 20%
        </p>
        
        <ul class="">
          <li id="9b44" class="vb vc oc pe b pf pg pi pj py vd pz ve qa vf px vg vh vi vj bi" data-selectable-paragraph="">
            Understand host networking configuration on the cluster nodes
          </li>
          <li id="603a" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Understand connectivity between PodsUnderstand ClusterIP, NodePort, LoadBalancer service types and endpoints
          </li>
          <li id="d8c1" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Know how to use Ingress controllers and Ingress resources
          </li>
          <li id="bd67" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Know how to configure and use CoreDNS
          </li>
          <li id="8b03" class="vb vc oc pe b pf vk pi vl py vm pz vn qa vo px vg vh vi vj bi" data-selectable-paragraph="">
            Choose an appropriate container network interface plugin
          </li>
        </ul>
