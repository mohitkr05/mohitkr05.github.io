Setting up my new PC with Ubuntu as my primary development workstation

Apr 15, 15158
9 minute read

Namaste! In this article, I will be sharing my experience of setting up my PC with Linux for development purposes. As a developer, I have always been fascinated by the open-source community and the flexibility it offers.

While Linux has been my primary operating system of choice, I have also used Windows and Mac OS from time to time for specific purposes. However, my love for Linux has always drawn me back to this open-source operating system. The flexibility, security, and control that Linux offers have always appealed to me as a developer. In fact, I have found that Linux has been instrumental in improving my productivity and efficiency as a programmer. With its vast array of tools and resources, Linux has enabled me to take on complex coding projects with ease. In this article, I will be discussing my recent decision to switch back to Linux as my main operating system for development purposes, and the steps I took to set up my PC for Linux.

In this article, I will be discussing the reasons behind my decision, the steps I took to set up Linux on my PC, and the benefits that I have experienced since the transition. Whether you are a seasoned developer or just starting out, I hope that my experience will inspire you to explore the world of Linux and its many advantages. So, let’s dive in and get started!

(PS! — I am continuously updating this list)

Browsers

Firefox

Firefox has been my preferred browser for many years now. The open-source philosophy of Firefox, coupled with its robust security features and customizable interface, have always appealed to me. As a developer, I rely heavily on various web development tools, and Firefox offers a plethora of add-ons that cater to my needs. Whether it’s the Web Developer add-on for debugging and analysing websites, or the Firebug add-on for inspecting HTML, CSS, and JavaScript in real-time, Firefox has always been my go-to browser for web development.

One of the things I appreciate most about Firefox is its commitment to user privacy. The browser comes with built-in privacy features that help protect my online identity and sensitive information from third-party trackers and hackers. Moreover, Firefox is constantly updated with the latest security patches, ensuring that I can browse the web without any worries.

Another feature that I find particularly useful is Firefox’s ability to sync my bookmarks, passwords, and browsing history across different devices. As someone who works on multiple computers and devices, this feature has been a lifesaver, allowing me to seamlessly access my bookmarks and saved passwords from anywhere.

Overall, Firefox has been an integral part of my web development workflow, and I would highly recommend it to anyone looking for a secure, customizable, and privacy-focused browser.

The best part it is included in the Ubuntu package.

Chromium

While Firefox is a powerful and versatile browser, there are some limitations that may has forced me to use Google Chrome instead. One such limitation is the lack of support for some websites and web applications that are optimized specifically for Google Chrome or Chromium. This is because some web developers may focus their efforts on optimizing their sites for Chrome, which can result in compatibility issues with other browsers like Firefox.

Another limitation of Firefox is that it may consume more system resources than Google Chrome, particularly when running multiple tabs or extensions. This can result in slower performance and longer load times, especially on older or less powerful computers.

Source code management (SCM)

Git

Git is a distributed version control system that is both free and open-source, and it is capable of handling projects of varying sizes — from small to very large — with exceptional speed and efficiency. Git is user-friendly and lightweight, with outstanding performance 

I mostly use Git with GitHub as my primary source code management system.Git is already installed with most of the Ubuntu systems but you can use the apt package manager if you want to install git.

Notes taking

Obsidian

Obsidian is a powerful note-taking and knowledge-management tool. As someone who regularly takes notes and conducts research, I have always been on the lookout for a tool that can help me organize my thoughts and ideas more effectively. That’s where Obsidian comes in — it’s a feature-rich, cross-platform note-taking application that allows you to create and organize notes in a way that makes sense to you.

The biggest advantage of using obsidian is that it is abile to create a network of interconnected notes, making it easy to navigate and discover new connections between ideas. Using the application’s graph view, you can visualize the connections between your notes, helping you to identify patterns and relationships that might not be immediately apparent.

Obsidian can be installed as a snap package in Ubuntu.

Joplin

Joplin is a popular open-source note-taking and to-do application gaining traction in recent years. With its intuitive user interface, cross-platform compatibility, and powerful features, Joplin has become a favourite among users looking for a reliable and customizable note-taking tool.

One of the unique features of Joplin is its support for end-to-end encryption, which allows you to protect your notes and other sensitive information from prying eyes. This feature is particularly valuable for users who need to store confidential or sensitive data, such as passwords, financial information, or personal details.

Another advantage of Joplin is its support for Markdown formatting, which enables users to create richly formatted notes that are easy to read and organize. With support for tags, notebooks, and the ability to create custom templates, Joplin makes it easy to organize and manage your notes in a way that makes sense to you.

Joplin also offers a range of synchronization options, allowing you to store your notes on your local machine, on a remote server, or in the cloud. This flexibility makes it easy to access your notes from anywhere, on any device.

Video recording & Editing

OBS

OBS — Open Broadcaster Software, a popular and powerful live streaming and video recording software used by content creators around the world. Whether you’re a gamer, musician, or just someone looking to share your thoughts and ideas with the world, OBS offers a range of features and tools to help you create and share high-quality content.

One of the things that sets OBS apart from other live streaming and video recording software is its open-source nature. This means that the software is free to use and can be customized and modified by the community to suit their specific needs. It also means that the software is constantly evolving and improving, with new features and updates being added all the time.

In addition to its open-source nature, OBS is also incredibly versatile and customizable. Whether you’re streaming to Twitch, YouTube, or another platform, OBS offers a range of settings and options to help you optimize your stream and ensure the best possible quality. From adjusting your bitrate and encoding settings to setting up custom scenes and transitions, OBS gives you full control over your stream.

SimpleScreenRecorder

SimpleScreenRecorder is a Linux program that was created by Maarten Baert to record programs and games.

Maarten Baert’s original goal was to create a program that was just really simple to use, but he was writing it he started adding more and more features. The result is actually a pretty powerful program. It’s ‘simple’ in the sense that it’s easier to use than ffmpeg/avconv or VLC, because it has a straightforward user interface.

Shotcut

Charlie Yates, the co-founder of MLT and the original lead developer, first came up with the idea for Shotcut in November 2004 (see the original website). Since then, the software has undergone a complete rewrite by another MLT co-founder and its current lead developer, Dan Dennedy. 

Dan’s vision for Shotcut was to create a new video editor based on MLT that could take advantage of the latest cross-platform capabilities, as well as the powerful WebVfx and Movit plugins. While Dan chose to reuse the name Shotcut due to his fondness for it, the current version of the software is a complete rewrite of the original.

snap install shotcut --classic

KdenLive

Kdenlive is a video editor that operates on open-source principles. Its development was initiated around 2003, and it is constructed on Qt and the KDE Frameworks libraries. The majority of video processing is executed through the MLT Framework, which relies on a range of other open-source initiatives such as FFmpeg, frei0r, movit, ladspa, sox, among others.

Kdenlive is designed to cater to a diverse range of video editing requirements, from fundamental to professional-level tasks.

To download and install Kdenlive, use the apt package manager

IDE & Text editors

VS Code

Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.js and has a rich ecosystem of extensions for other languages and runtimes (such as C++, C#, Java, Python, PHP, Go, .NET)

To install VS code download the package and install it. You can use the dpkg command to install the package manager.

Vim

Vim is a highly configurable, open-source text editor that is known for its speed, efficiency, and versatility. Originally released in 1991, Vim is a descendant of the popular Vi editor and is designed to work in a terminal environment, making it ideal for use on servers or other systems without graphical interfaces.

One of the things that sets Vim apart from other text editors is its extensive range of keyboard shortcuts and commands. These shortcuts allow experienced Vim users to navigate, edit, and manipulate text with incredible speed and precision, often without ever having to take their hands off the keyboard.

Another key feature of Vim is its high level of configuration. Users can customize the editor to suit their specific needs and preferences, from changing the appearance of the interface to defining their own key mappings and commands.

Despite its initial learning curve, Vim has developed a passionate following among developers, sysadmins, and other power users who value its speed, efficiency, and customizability. Whether you’re working on a remote server or just looking for a powerful text editor that can keep up with your workflow, Vim is definitely worth considering.
